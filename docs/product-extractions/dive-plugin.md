# Product Extraction: DIVE for Codex

**Decision date**: 2026-07-15

DIVE for Codex is an independent product rather than a DIVE desktop feature.
Its local repository is:

`~/dive-plugin`

The new repository owns its product constitution, Codex plugin manifest,
feature specifications, plans, tasks, implementation, verification, releases,
and marketplace distribution. DIVE-2 remains authoritative only for the DIVE
desktop application.

The products share educational principles — real-project supervision,
evidence-grounded intervention, student-owned decisions, separation of AI
self-report from verification evidence, Work/Guided modes, and local-first
research records — but do not share runtime guarantees. In particular, the
Codex plugin does not inherit DIVE's Pi/Rust supervision boundary or custom
desktop UI.
