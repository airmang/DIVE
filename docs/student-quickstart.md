# 학생용 빠른 시작 (5분 가이드)

## DIVE가 뭐야?

**DIVE(Decompose, Instruct, Verify, Extend)** 는 AI와 함께 코드를 **안전하게** 만드는 수업 도구입니다. 그냥 "AI야 만들어줘"가 아니라, **4단계**를 거쳐 너가 직접 생각하고 확인하면서 진행합니다.

- **D (Decompose)** — 만들 것을 **작은 카드**로 나눕니다.
- **I (Instruct)** — 각 카드에 **무엇을 해야 하는지 지시**를 적습니다.
- **V (Verify)** — AI가 만든 결과가 지시와 맞는지 **검증**합니다.
- **E (Extend)** — 검증이 끝난 코드를 **확장·개선**합니다.

## 1분: DIVE 실행

1. 바탕화면에서 **DIVE** 아이콘 더블클릭
2. "Windows가 PC를 보호했습니다" 창이 뜨면 → **추가 정보 → 실행**
3. 앱이 열리면 **온보딩 창** 표시

## 2분: 프로바이더 연결

1. 온보딩 창에서 선생님이 알려준 **프로바이더** 선택 (보통 OpenRouter)
2. 선생님이 준 **QR 코드**를 스캔 또는 **API 키**를 직접 입력
3. **[연결하기]** 클릭
4. 오른쪽 위에 초록 점 (●) 표시되면 성공

## 3분: 프로젝트 + 세션 만들기

1. 왼쪽 사이드바 **+ 새 프로젝트** 클릭
2. 이름 입력 (예: "내 첫 웹페이지")
3. 폴더 경로 입력 (예: `C:/Users/내이름/Desktop/dive-first`)
4. **프로젝트 생성** 클릭
5. 사이드바 **+ 새 세션** 클릭 → 자동으로 세션 생성

## 4분: D 단계 — 카드 만들기

카드 없으면 채팅이 안 됩니다 (안전 장치). 두 가지 방법:

### 방법 A: AI 도움 받기 (추천)

1. 하단 워크맵 띠에서 **"AI 도움 받기"** 버튼 클릭
2. 무엇을 만들고 싶은지 한 줄로 입력 (예: "로그인 페이지 만들어줘")
3. AI가 4개 카드 초안 제안 → 마음에 드는 것 체크 → **모두 수락**

### 방법 B: 직접 카드 추가

1. 워크맵 **"+ 카드 추가"** 클릭 (D 단계에서만 활성)
2. 카드 제목 입력 (예: "이메일 입력 칸 만들기")

## 5분: I → V → E

1. 만든 카드 클릭 → 카드 상세 패널 열림
2. **지시 작성** 칸에 구체적인 요구 입력 (예: "placeholder는 example@email.com, 빨간 테두리 X")
3. 채팅창에 지시 반복 → AI가 코드 작성
4. AI가 파일을 바꾸려 하면 **권한 카드** 팝업 → **diff** 확인 후 **승인**
5. 지시대로 됐다면 **[검증 요청]** → V 상태 → **[승인]** → ✅ verified

## 팁

- **Ctrl+S** 눌러 언제든 수동 체크포인트 저장
- 타임라인의 **초록 점**은 방금 verified된 상태 — 언제든 돌아갈 수 있음
- AI가 엉뚱한 답을 하면 **[다시 시도]** 또는 채팅에 "다시 해줘" 입력
- 막히면 선생님에게 손 들기

## 안전 규칙

권한 카드는 **내가 판단하고 허용하는** 3단계입니다:

- **Safe(자동 실행)** — 읽기 전용 도구(read_file, list_dir, search_files)는 AI가 자동 실행
- **Warn(변경 — diff 확인 후 허용)** — 파일을 바꾸는 도구(write_file, edit_file)는 **바뀔 내용(diff)을 확인한 뒤 내가 허용**
- **Danger(삭제·명령 실행 — 내가 직접 판단해 허용)** — 파일 삭제·프로세스/터미널 실행(delete_file, run_process, run_terminal_script)은 **내가 직접 판단해서 허용** (자동으로 막히지 않아요)

정말 파괴적인 명령(`rm -rf`, 시스템 파일 수정 등)만 DIVE가 **자동 차단** — 권한 카드에 "차단됨" 표시.

## 차시 끝나면

1. 사이드바에서 세션 선택
2. **세션 내보내기** 버튼 (또는 메뉴)
3. `.jsonl` 파일 저장 → 선생님에게 USB로 제출 또는 공유 폴더 업로드

---

뭔가 안 되면 "F1" 키나 선생님. **실수해도 체크포인트로 복원 가능**하니 겁먹지 마!
