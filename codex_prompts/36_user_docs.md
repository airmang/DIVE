   - 일반: "DIVE는 무엇인가?", "왜 DIVE를 쓰나?", "ChatGPT랑 뭐가 다른가?"

   - 권한: "권한 카드를 매번 승인하는 게 귀찮아요", "자동 승인 가능한가?"
   - DIVE 단계: "꼭 D부터 해야 하나?", "AI 도움 없이 카드 만들 수 있나?"
   - 데이터: "내 코드가 어디로 가나?", "오프라인 사용 가능한가?"
   - 학생: "학교 계정으로 사용?", "수업용 키 어떻게 받나?"

3. **트러블슈팅** (`docs/troubleshooting.md`):
   - 설치 실패 — SmartScreen, WebView2, ARM64
   - 네트워크 에러 — 학교 방화벽, 프록시
   - 키 인증 실패 — 만료, 한도 초과
   - 빌드 실패 (코드 변경 시) — Rust 의존성, ARM64
   - 미리보기 안 보임 — iframe 차단, 로컬 서버 충돌
   - 체크포인트 복원 후 워크맵 상태 이상
   - 각 항목 — 증상 / 원인 / 해결 단계
4. **글로서리** (`docs/glossary.md`):
   - DIVE, Decompose, Instruct, Verify, Extend
   - 카드, 워크맵, 세션, 프로젝트
   - 권한 카드 (안전·주의·위험)
